# Patient Risk Scoring - Breast Cancer Detection

## 🎯 Objectives
* Prepare and split clinical datasets for risk prediction.
* Train a **Random Forest Classifier** to assess patient risk.
* Evaluate clinical relevance using Sensitivity (Recall) and Accuracy.

## 📊 Dataset
The model uses the `breast_cancer.csv` dataset, which contains clinical features like mean radius, texture, and perimeter of cell nuclei.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-learn, Matplotlib, Seaborn
* **Model:** Random Forest (Supervised Learning)

## 📈 Performance Highlights
The model focuses on **Sensitivity** to ensure medical risks are not overlooked, achieving high performance across standard classification metrics.
